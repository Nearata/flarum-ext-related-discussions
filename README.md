# Related Discussions

A [Flarum](http://flarum.org) extension. No description.

## Install

```sh
composer require nearata/flarum-ext-related-discussions:"*"
```

## Update

```sh
composer update nearata/flarum-ext-related-discussions:"*"
php flarum migrate
php flarum cache:clear
```
